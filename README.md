# Dhaka-Dash-3D-Racing-Car-Game
Dhaka Dash is a fast-paced 3D endless runner shooter set on a chaotic highway. Dodge obstacles, switch lanes, battle enemy vehicles, collect treasures, and use powerful weapons while speed constantly increases. Choose first- or third-person view and survive as long as you can.

Game Controls – Dhaka Dash

Movement

⬅️ Left Arrow – Move to the left lane

➡️ Right Arrow – Move to the right lane

Combat

I or Right Mouse Button (RMB) – Fire weapon

K or Left Mouse Button (LMB) – Launch bomb attack

Camera & Modes

V – Switch between First-Person and Third-Person view

G – Toggle God Mode (invincibility)

Menu & Navigation

1 / 2 / 3 – Select Easy / Normal / Hard mode

TAB – Start the game

Q – Return to main menu after game over
