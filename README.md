# Dhaka-Dash-3D-Racing-Car-Game
Dhaka Dash is a fast-paced 3D endless runner shooter set on a chaotic highway. Dodge obstacles, switch lanes, battle enemy vehicles, collect treasures, and use powerful weapons while speed constantly increases. Choose first- or third-person view and survive as long as you can.
